## Laravel Marketplace

Marketplace dengan menggunakan framework laraver dan machine learning dengan metode kmeans untuk clustering penjual

## Fitur

- _CRUD produk, kategori, atribute
- _Slide 
- _Blog
- _Ongkir
- _Cart & Order sistem
- _User & Role
- _Email notif
- _Admin dashboard & user dashboard
